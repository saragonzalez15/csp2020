Busy Bee

In this project, we culminated all the knowledge we learned throughout the unit 1.1. Our project displays a scene that consists of multiple flowers, a bee, and a beehive. We created these components by finding images online and creating them into GIFs in order to import them into Python Turtle. The flower was created using a for loop in order to shorten the process of importing and positioning six different flowers. In order to get the bee to move we used a while loop that will repeat six times. This allows the bee to fly from the hive, down to all the flowers, and collect the honey to then bring back up to the hive.

Shark Time

In this project, we utilized all the programming skills we learned in the unit 1.2 to create a game similar to Pac-Man. We used GIFs for the shark, the fishes, and the underwater background. We coded the shark's movement so that the user could move the shark using the computer's arrow keys. The maze's walls, barriers, and the fishes were coded to be placed randomly each time the game is loaded. Everytime the shark eats a fish, the score increments by 1. If the shark runs into the wall it flips over, dies, and the game is over. However, if you are able to eat all ten fish, the game ends and the screen displays a message of "You Win".
